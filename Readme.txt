================================================================================
1. はじめに
================================================================================

このたびは「EbIRC」をダウンロードしていただきありがとうございます。
このソフトウェアは、Windows Phone 6 シリーズ以降で動作する IRC クライアントです。

================================================================================
2. 動作環境と確認環境
================================================================================

EbIRC の動作には以下のようなシステムが必要です。

・Windows Mobile 6 Professional 以降を搭載した Windows Phone
・.NET Compact Framework 3.5 が導入済みであること

以下の端末で動作を確認しました
・docomo PRO series T-01A (TOSHIBA TG01)
・WILLCOM Advanced/W-ZERO3[es] WS011SH

================================================================================
3. 使用方法
================================================================================

まず、EbIRC のパッケージをインストールしてください。
プログラムの一覧に EbIRC が登録されます。
(デバイス上で cab ファイルを開くとインストールできます)

まず、設定画面を開いて、接続タブで接続に必要な情報を入力してください。
次に、左ソフトキー「接続」を押してください。サーバーへ接続します。
